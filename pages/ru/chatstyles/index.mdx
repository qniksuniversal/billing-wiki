Стили чата позволяют изменить внешний вид ваших сообщений в игровом чате. Никакого влияния на геймплей — только красота.

### Доступные стили

Каждому игроку по умолчанию доступны **3 стиля**:

Игрокам с подпиской **Schalker** доступны все **7 подготовленных стилей**.

### Как выбрать стиль

Откройте меню выбора стиля командой <code>/chat</code> — выберите понравившийся вариант из доступных. Стиль применяется только сообщениям которые видите вы и не виден всем игрокам в чате.

![chat_styles](chatstyles/chat1.png)

### Создание своего стиля

Обладатели подписки **Schalker** могут создать полностью **собственный стиль чата**:

1. Зайдите на сайт сервера и откройте свой **профиль**.
2. В блоке с активной подпиской нажмите кнопку **«Стили чата»**.
![chat_styles](chatstyles/chat2.png)
3. Настройте оформление по своему вкусу.
4. Сохраните — стиль появится у вас в списке доступных.
