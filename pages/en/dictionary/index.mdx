## Technical terms
### TPS
TPS (abbreviation for English <code>Ticks per Second</code>) - number of ticks per second.
The higher this indicator, the more productive the server is.
The normal TPS in the game and on the server is 20.0.
Calculated from the ratio of 1000 milliseconds to the average time it took to complete 1 clock cycle, calculated over the last 20 clock cycles.
To obtain a normal indicator of 20.0, it is required that the average rendering time of 1 clock cycle does not exceed 50 milliseconds.
Description taken from [unofficial Minecraft Wiki](https://minecraft.fandom.com/ru/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%BE%D0%B2#%D0%A2)

### FPS
FPS (short for English <code>Frames per Second</code>) - number of frames per second.
The higher this indicator, the “smoother” the image on the screen changes.

### Chunk
Chunk (from English <code>chunk</code>) is a unit of game space.
The dimensions of one chunk are 16 blocks wide, 16 blocks long and from the very bottom to the very top of the world.
Description taken from [unofficial Minecraft Wiki](https://minecraft.fandom.com/ru/wiki/%D0%A1%D0%BF%D0%B8%D1%81%D0%BE%D0%BA_%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%BE%D0%B2#%D0%A7)

## General
### Tab
The tab is the screen that appears when you press the <kbd>Tab</kbd> button on your keyboard.
This screen displays a list of players, as well as basic information about the server you are on: name, online
