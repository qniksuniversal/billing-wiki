An item used to edit block states. Looks like a breeze rod, but is a debugging stick with all its capabilities.

## Craft

- **Breeze Rod** × 2
- **Star of Nether** × 1

![debug_craft](debug-stick/debug_stick_craft.png)

## Usage

Hitting a block with a stick allows you to select its editable state; for example, you can change the editable state from `conditional` (conditional) to `facing` (direction) of the command block. Using a stick on a block allows you to change the value of the editable state; for example, you can change the command block's direction to south by changing the state value `facing` to `south`. If you sit down, the order of changing states/values ​​will change in the opposite direction.

## Restrictions

The debug wand **doesn't work** with some blocks. If you try to use it on such a block, a warning will appear.

Changing the state `waterlogged` (flooding with water) **prohibited** for all blocks.
