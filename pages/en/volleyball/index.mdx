With this plugin you can feel like you are in a physical education lesson. The ball can be thrown, hit, served, in general, whatever you need. The plugin contains physics and gravity.

<img src="/wiki/volleyball/volleyballcraft.png" width="818" height="246" />

## Usage

| Button | Destination |
| --- | --- |
| <kbd>Q</kbd> | Throw the ball with minimal force |
| <kbd>RUN + LMB</kbd> | Throw the ball harder |
| <kbd>SHIFT + LMB</kbd> | Toss the ball on the spot |
| <kbd>RMB + EMPTY HAND</kbd> | Pick up the ball |
| <kbd>PKM</kbd> | Place the ball |
