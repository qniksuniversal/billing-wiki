Do you want your sword to be called not “Iron Sword”, but “Blade of Dawn” with a golden gradient and three lines of description? It's here.

### Item name

Take the item in your hand and enter the command:

<code>/itemname <text></code> — set the name.

<code>/itemname clear</code> - reset to standard.

![item](itemname/item.png)

### Item lore

Lore is a description under the name of an item. Supports multiple lines.

<table>
    <thead>
        <tr>
<th>Team</th>
<th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
<td><code>/itemlore add <text></code></td>
<td>Add a new line to the description.</td>
        </tr>
        <tr>
<td><code>/itemlore set <number> <text></code></td>
<td>Replace a specific description line.</td>
        </tr>
        <tr>
<td><code>/itemlore remove <number></code></td>
<td>Delete line by number.</td>
        </tr>
        <tr>
            <td><code>/itemlore clear</code></td>
<td>Clear all description.</td>
        </tr>
    </tbody>
</table>

### Color design

Colors and gradients are supported in the title and lore. Gradient example:

<code><gradient:#F4FF31:#C18009>Ax ComboGames_GG</gradient></code>

To select beautiful gradients, use the generator:
[birdflop.com/resources/rgb](https://www.birdflop.com/resources/rgb/)

### Nice small font

Do you want to add an exquisite touch to the item's lore? Use the Small Caps generator:
[ru.piliapp.com/cool-text/small-caps](https://ru.piliapp.com/cool-text/small-caps/)

Copy the result and paste it into the command - it looks like an artifact from another dimension.
