A dynamic map is an interactive server web map that updates in real time. See the terrain, buildings and player positions directly from the browser.

> Available **only for Schalker subscribers**.

### How to open a card

Follow one of the links:

- [schalker.net/map](https://schalker.net/map) - built-in map on the site.
- [map.schalker.net](https://map.schalker.net/) - map in full screen mode.

### Visibility on the map

By default, you are shown on the map. If you don't want to be seen, hide as a team.

<th>Team</th>
