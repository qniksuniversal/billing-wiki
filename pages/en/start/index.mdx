Ketrix is a universal hosting provider that offers digital solutions for clients of different sizes. We work worldwide and use this wiki to collect answers, instructions, and large guides for installing and configuring software.

This page explains why it is worth using Ketrix instead of looking for someone else.

## Why Ketrix

We provide virtual services based on **AMD Ryzen 9 9950X3D**, fast **NVMe drives** with speeds up to **7000 MB/s**, and network connectivity up to **10 Gbit/s**.

Servers are protected by powerful DDoS filtering up to **2 Tbit/s**. We also do not build the service around overselling: at any moment you can load your server at **100% of your plan capacity**.

## Payments, taxes, and crypto

We pay taxes and accept payments in almost all world currencies. Cryptocurrency payments are supported too.

All servers are located in our own mini data center. This matters because we can control infrastructure, hardware, maintenance, and service quality more closely.

### Do we use swap?

No. We do not use swap on your services and do not sell it as a separate paid option. If you buy a plan, you should receive the real resources of that plan.

We try not to limit the platform to only one type of client.

For Minecraft, you can quickly install the required mods, plugins, and builds. For MTA, ready-made builds are available in a separate panel tab and can be installed completely free.

For web projects, you can install a ready-made forum or another template. It will be configured automatically, and you will only need to fill in administrator data and upload a logo.

## VDS

VDS also has a convenient control panel. In a couple of clicks you can:

- reinstall the server;
- request a plan upgrade;
- configure firewall rules;
- check server load;
- manage basic server settings.

If you need one VDS for a website, game project, and a few bots, that is a normal use case. You can keep everything in one place and enable additional protection when needed.

## Server protection

All servers include basic **L3/L4** DDoS protection.

If a project needs stronger protection, you can additionally purchase **L7 protection**. It is available for VDS and works on all ports.

This is useful when not only the website is under attack, but the whole project: game server, website, bots, and other services.
